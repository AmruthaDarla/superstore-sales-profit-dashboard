# Superstore Sales & Profit Analysis Dashboard

## Project Overview
This project analyzes Superstore retail sales data using SQL Server and Excel. The goal was to identify sales performance, profitability trends, top product categories, and regional business insights.

## Tools Used
- SQL Server
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Dashboard

## Dataset
The dataset contains 9,994 retail transaction records with fields such as category, region, sales, quantity, discount, and profit.

## Key Insights
- Technology generated the highest sales.
- West region was the most profitable.
- Phones led sub-category revenue.
- Central region underperformed in quantity.
